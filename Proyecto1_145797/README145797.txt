READ ME:

Para ejecutar el programa “Proyecto1”,
 seguir los siguientes pasos:

1. Activar ROS master  con el uso del comando roscore y mantener dicha terminal abierta
2. Abrir una nueva terminal y correr turtlesim
rosrun  y turtlesim turtlesim_node

3. Descargar la carpeta adjunta “Aperezbar”.

4. Abrir otra terminal y posicionarse en el directorio Aperezbar 

5. Compilar el paquete con ayuda del comando 
catkin_make

6. ejecutar el comando source devel\setup.bash para asegurarse de encontrar el paquete deseado
7. Correr el programa desde la terminal con 
rosrun proyecto1 Proyecto1
8. Seguir las instrucciones ,dando la cordenada X y Y a la que desee moverse ,
9.posteriormente el programa le solicitara el angulo en el que desea que termine la tortuga
10. el programa le pedira una aproximacion del tiempo en el que se desee que se realice 
11.Para Ingresar cada dato basta con presionar la tecla enter

12.Al terminar la accion ,el programa solicitara si desea continuar, en caso afirmatibo presione 1 para ingresar una nueva posición.
